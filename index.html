LLM-Powered Nuclei Template Generator & Scanner

LLM連携型 Nucleiテンプレート 自動生成 ＆ スキャナー

Generate Nuclei-compatible YAML vulnerability scanning templates from natural language using Gemini API, and execute quick security verification tests against your targets immediately.

Gemini APIを活用し、自然言語のプロンプトからNuclei互換のYAMLテンプレートを自動生成。その場ですぐにターゲットに対して脆弱性スキャンのテストを実行できます。

Core Features

コア機能

AI-Driven Generation

AIテンプレート自動生成

Describe what you want to test in natural language, and Gemini API will instantly write the corresponding YAML Nuclei templates.

「テストしたい脆弱性の挙動」を日常的な日本語や英語で記述するだけで、Gemini APIが即座に対応するYAMLスキャンテンプレートを作成します。

Safe Direct API Connection

セキュアな直結通信

Direct HTTPS connection to Google Gemini API. No third-party proxy required. Rest assured your API keys and targets are kept private.

サードパーティのプロキシを経由せず、アプリからGoogle Gemini APIに直接接続。セキュリティ監査に必要なAPIキーや通信情報が漏洩しません。

On-the-Fly Scanning

即時テストスキャン

Execute the generated template on your target host with a single click. View real-time status and extract matching payload info.

生成されたYAMLテンプレートを即時ターゲットに実行。レスポンスのチェックや正規表現によるマッチ情報をリアルタイムに出力します。

Lightweight Rust Native

超軽量 Rust ネイティブ

Built with Rust and egui. Native performance without the bloat of web technologies like Electron. Super memory-efficient.

Rustおよびeguiによるネイティブ動作。Electronのような重いエンジンを持たず、起動も一瞬でメモリをほぼ消費しません。

Quick Start Guide

クイックスタートガイド

1

Compile and Run the App

アプリをビルドして実行する

Clone the repository and compile it with Cargo.

リポジトリをクローンし、Cargoを使用してビルド・実行します。

# Clone and build
git clone https://github.com/olbin-dev/logos-cyber.git
cd logos-cyber
cargo run --release
2

Set Gemini API Key & Prompts

Gemini APIキーとプロンプトの設定

Enter your Gemini API key in the configuration panel. Describe your test target URL and prompt in natural language.

アプリのサイドパネルにGemini APIキーを入力し、テスト対象のURLと「テストしたい脆弱性内容」をプロンプトに入力します。

3

Generate and Scan

テンプレートの生成とテスト実行

Click "Generate with AI" to generate the YAML template. Review the code, then click "Run Quick Scan" to run the audit against the target URL.

「Generate with AI」をクリックしてテンプレート（YAML）を出力させ、内容を確認後「Run Quick Scan」でターゲットに対する脆弱性監査を実行します。
